====================================
Welcome to the NanoCD documentation!
====================================
This is the documentation of Nano CD project (which is based on
`Python For the Lab <http://www.pythonforthelab.com>`_).  It is the
software to control the nanoCD experiment at nano-optics
(`Kuipers Lab <http://kuiperslab.tudelft.nl/>`_)
group at QN, TU Delft.
